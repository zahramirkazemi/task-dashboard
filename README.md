## technologies
React - Zustand - Typescript - Rspack - Ant design

todo api: https://jsonplaceholder.typicode.com

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
